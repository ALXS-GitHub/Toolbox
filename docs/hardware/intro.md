# Hardware

This section documents my current personal setup. The foundation is an ASUS TUF Gaming F17 laptop that serves as my main machine for both development and gaming. Around it sits the set of peripherals I use daily: a hall effect mechanical keyboard, an ultra-lightweight wireless mouse, and a tri-mode wireless headset. Each page covers the hardware, why it was picked, and how it fits into the workflow.
