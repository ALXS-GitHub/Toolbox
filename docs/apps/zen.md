# Zen

Zen is a web browser based on Firefox. It is such a nice browser as it has this left sidebar which is super convenient for multitasking and organizing tabs, and this is why I like it very much.

## Syncing

For syncing tabs and setup, Zen provides a built-in account system that allows you to sync your settings and open tabs across devices.
