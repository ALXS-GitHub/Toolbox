# Powershell Config

Voici tous les outils personnalisés de mon terminal

## Afficher l’aide

```bash
myhelp
```

## Configuration

Premièrement dans la configuration, on a les couleurs du promtp. Elles peuvent être trouvées dans l’application terminal, puis settings puis ouvrir le fichier JSON (en bas à gauche). Ici on peut ajouter des nouveaux thèmes de couleurs pour la partie prompt.

Le reste de la configuration (thème, commandes custom, …) se retrouvent dans `C:\Users\alxsm\.config\powershell` .

## Thèmes

Pour chaque thème (fond d’écran, et / ou utilisation de Warp), il est conseillé de faire un nouveau fichier de theme (.omp.json) avec des couleurs appropriées pour ce thème. Je conseille [https://coolors.co/image-picker](https://coolors.co/image-picker) qui permet d’avoir une palette de couleur en fonction d’une image.
