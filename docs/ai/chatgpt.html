ChatGPT

ChatGPT is OpenAI's flagship conversational AI, built on the GPT (Generative Pre-trained Transformer) architecture. It handles natural language conversations across a vast range of tasks: answering questions, drafting and editing text, brainstorming ideas, explaining concepts, writing and debugging code, summarizing documents, and creative writing.

Key capabilities​

Multimodal input​

ChatGPT accepts text, images, files (PDFs, spreadsheets, code), and voice. Upload screenshots for analysis, share documents for summarization, or have spoken conversations.

Custom GPTs​

Build specialized assistants by combining instructions, knowledge files, and optional API actions. Create domain-specific helpers (legal review, code review, tutoring) that can be shared or kept private.

Canvas​

A collaborative workspace for writing and code. Canvas displays content in an editable side panel where both you and ChatGPT can make inline modifications, track changes, and iterate on drafts together.

Memory & personalization​

ChatGPT stores facts about you (preferences, work context, prior projects) across sessions, making responses more relevant over time. Memory can be reviewed and cleared in settings.

Web browsing & search​

ChatGPT searches the web for current information, cites sources, and answers questions about recent events beyond its training cutoff.

Code execution & data analysis​

Upload CSVs, Excel files, or datasets and ChatGPT runs Python code in a sandboxed environment to analyze, visualize, and transform data.

Available models​

GPT-5 Series​

The flagship models for complex tasks. Offers three modes: Auto (recommended), Fast, and Thinking for deeper reasoning. Significantly reduced hallucination compared to GPT-4.

Reasoning models (o-series)​

  • o3: Frontier reasoning for coding, math, science, and visual perception
  • o4-mini: Fast, efficient reasoning optimized for math and coding
  • o3-pro: Extended thinking for maximum reliability

GPT-4o​

Multimodal model with text, images, files, and voice. Good balance of speed and capability.

Practical tips​

  • Choose the right mode: Use Thinking mode for complex problems, Fast for quick queries
  • Be specific: "Summarize this PDF focusing on the methodology section" yields better results than "summarize this"
  • Iterate: Refine outputs by asking for changes ("make it more concise", "add code examples")
  • Use system prompts in Custom GPTs: Define behavior, constraints, and persona upfront
  • Check citations: When using web browsing, verify linked sources

Limitations​

  • Can produce plausible but incorrect information (though reduced in newer models)
  • Long conversations may lose early context
  • Complex multi-step reasoning can still fail; verify critical outputs
  • Rate limits apply based on subscription tier

Recent updates​

GPT-5.2 (December 2025) introduced context compaction for longer tasks, improved vision capabilities, and is the first model to cross 90% on ARC-AGI-1 benchmarks.

Summary​

ChatGPT is a versatile general-purpose AI assistant for writing, research, coding help, data analysis, and everyday questions. Custom GPTs and Canvas extend it into a platform for building specialized tools.
