# Markdown Theme VSC Extension

Extension markdown associée au repository Markdown Themes

## Mise en place

Pour que les thèmes soient utilisables dans l’extension vscode, il faut déjà télécharger le repo `Markdown Themes`, puis il faut setup la variable d’environnement suivante `ALXS_MD_THEME_PATH=<path_to_repo>\Markdown-Themes\themes`

Cette extension utilise aussi le repo Markdown-to-PDF. Vérifier que le submodule est à jour
