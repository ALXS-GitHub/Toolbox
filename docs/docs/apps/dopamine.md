# Dopamine

**[Dopamine](https://github.com/digimezzo/dopamine)** is a music player for people who want something clean and simple for their local library. Built with Electron and Angular, it has a polished, minimalist interface with smooth animations that stays out of the way and lets you focus on listening. It organizes your collection by artist, album, genre, and folder, pulling everything it needs from your files' embedded metadata (ID3 tags) -- so well-tagged music means a better browsing experience.

Dopamine runs on Windows, Linux, and macOS. On Windows and macOS, grab a build from the [GitHub releases](https://github.com/digimezzo/dopamine/releases) page or build from source (requires Node.js 22). On Linux, you can install it with Snap:

```bash
snap install dopamine
```

It is also available on the AUR as `dopamine-official`. The project is free and open-source under GPL-3.0.
