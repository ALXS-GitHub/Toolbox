# Surf

**[Surf](https://deta.surf/)** is a web browser that integrates AI features to enhance your browsing experience. It offers a clean and intuitive interface, focusing on a desktop-like experience motivated by the fact that our web browsers are the app that is the most used on our computers and that we should be able to find all documents, files, and web apps in one place.
