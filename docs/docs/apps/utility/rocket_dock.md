# RocketDock

**[RocketDock](https://punklabs.com/rocketdock)** is an application launcher for Windows that provides a clean, animated dock for quickly accessing your favorite programs and files. It is free and lightweight.

## Core Features

- **Application launcher**  
  Add shortcuts for frequently used programs, folders, and files to a visually appealing dock.  

- **Customizable icons**  
  Change icons and labels to match your preferred style and organization.  

- **Auto-hide and always-on-top options**  
  Keep the dock out of the way when not in use or display it on top of other windows.

## Additional Features

- **Drag-and-drop support**  
  Easily add shortcuts by dragging them onto the dock.  

- **Multi-monitor support**  
  Place the dock on any monitor in multi-display setups.  

- **Smooth animations**  
  Enjoy polished transitions and effects when interacting with dock icons.

## Icons

RocketDock allows you to customize the icons for your shortcuts, giving you the ability to change their appearance and make them more visually appealing. To do this, just put your icons in the `C:\Program Files (x86)\RocketDock\Icons` folder, and then you will be able to select them in the icon properties.

## Characteristics

- **Free and lightweight**  
  Minimal system resource usage while providing visual enhancements.  

- **Highly customizable**  
  Supports multiple themes, icon packs, and dock behaviors.  

- **Windows-only**  
  Designed specifically for Windows desktops.
