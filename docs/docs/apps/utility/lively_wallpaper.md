# Lively Wallpaper

**[Lively Wallpaper](https://www.rocksdanister.com/lively/)** turns your desktop background into something that actually moves. It ships with a handful of built-in animated themes -- rain, snow, clouds, synthwave -- and can use videos, GIFs, or web pages as wallpapers. The interface follows the Windows 11 design language and stays out of your way.

The detail that matters most: wallpapers pause automatically when a fullscreen application or game is running, dropping resource usage to near zero. You get the eye candy on your desktop without paying for it during gameplay. Multi-monitor and ultrawide setups are supported, and you can apply post-processing effects like depth adjustment, fog, or CRT filters to any wallpaper.

If you want to go further, Lively lets you create your own wallpapers using HTML, CSS, and JavaScript -- music visualizers, interactive widgets, live data displays, anything a browser can render. It is free, open-source (GPL-3.0), and available from the [Microsoft Store](https://apps.microsoft.com/detail/lively-wallpaper) or as a direct download from the [official site](https://www.rocksdanister.com/lively/).
