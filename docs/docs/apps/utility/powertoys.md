# PowerToys

[PowerToys](https://learn.microsoft.com/windows/powertoys/) is a Windows utility suite from Microsoft that provides small, focused tools to improve productivity and streamline power-user workflows. It is made for people who want to customize and speed up everyday tasks: window management, quick-launching apps, batch renaming/resizing files, picking colors from the screen, remapping keys, and more all accessible via a single app and with configurable shortcuts.

Here are some examples of my favorite tools and how to use them:

- **PowerToys Run**: Press `Alt+Space`, type an app, file name, or expression (e.g., "notepad", "calc 2+2") and press Enter.
- **Command Palette**: Recently the PowerToys Run tool had an update with a tool called "Command Palette". basically it is the same tool but with enhanced search and features. Press `Win+Alt+Space` to open it.
- **Image Resizer**: Select images in File Explorer, right‑click → Resize pictures, choose a size and apply.
- **Color Picker**: Press `Win+Shift+C`, hover/click to capture a color; the hex/RGB value is copied to clipboard.
- **Always On Top**: With a window focused, press `Win+Ctrl+T` to toggle it being always on top.
- **Shortcut Guide**: Press and hold the Windows key to show the overlay of common Win-key shortcuts.
- **Awake**: Toggle from the PowerToys tray menu or the Awake page in settings to keep the machine awake.
- **File Explorer add-ons**: Enable in PowerToys Settings → Explorer add-ons to preview images and files content in Explorer’s preview pane.

There are many other tools available in PowerToys.
