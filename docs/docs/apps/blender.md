# Blender

**[Blender](https://www.blender.org/)** is a free and open-source 3D creation suite used for modeling, animation, rendering, and more. I use it for creative projects, and it is widely adopted in both amateur and professional workflows.

## Core Features

- **3D modeling**  
  Create and edit 3D objects using mesh, sculpting, and procedural modeling tools.  

- **Animation and rigging**  
  Animate characters and objects using keyframes, armatures, and shape keys.  

- **Rendering**  
  Generate high-quality images and videos using built-in engines like Eevee and Cycles.

## Additional Features

- **Simulation**  
  Create physics-based effects like fluids, smoke, fire, cloth, and rigid body dynamics.  

- **Video editing**  
  Basic video editing and compositing within the same application.  

- **Scripting and automation**  
  Use Python scripts to extend Blender’s functionality or automate tasks.  

- **Add-ons and community extensions**  
  Install community-created add-ons to expand modeling, rendering, and workflow tools.

- **Cross-platform support**  
  Available on Windows, macOS, and Linux, fully open-source.

## Characteristics

- **Open-source and free**  
  Completely free to use, modify, and distribute, with a large and active community.  

- **Professional-grade tools**  
  Suitable for animations, visual effects, 3D printing, and game asset creation.  

- **Highly customizable**  
  Extensive options for interface layout, shortcuts, and add-ons to tailor the workflow.
