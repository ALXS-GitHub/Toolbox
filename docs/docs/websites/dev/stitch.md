# Google Stitch

**[Google Stitch](https://stitch.withgoogle.com/)** is a free AI-powered UI design tool from Google Labs. It generates high-fidelity user interfaces for web and mobile apps from text prompts, hand-drawn sketches, screenshots, or voice descriptions — and exports clean HTML/CSS code.

## Key features

### Multi-modal input

- **Text prompts** — describe what you want ("a settings page with dark mode toggle and notification preferences")
- **Image/sketch upload** — upload a wireframe or screenshot, Stitch generates a polished UI from it
- **Voice** — speak directly to the canvas, get real-time design critiques and updates as you talk
- **URL** — extract a design system from any existing website

### AI-native canvas

An infinite canvas where ideas go from early ideation to working prototypes:
- Generate multiple design variations from a single prompt
- Connect screens into interactive prototypes — click "Play" to preview the flow
- Stitch can automatically generate logical next screens based on your flow

### Manual editing (March 2026)

Click on any element to edit text, swap images, adjust spacing — fine-grained tweaks without re-prompting.

### Design systems (DESIGN.md)

Extract a design system from any URL or use `DESIGN.md` to export/import design rules across Stitch projects and other design/coding tools.

### Developer integration

A new SDK and MCP server connect Stitch to coding assistants like Claude Code, Gemini CLI, and Cursor for design-to-code workflows.

## Pricing

- **350 free generations/month** in standard mode
- Completely free during public preview

## Use cases

- Rapid prototyping of mobile/web app UIs
- Turning sketches or wireframes into polished designs
- Generating design variations for A/B testing
- Extracting design systems from existing sites
- Creating interactive prototypes without Figma
