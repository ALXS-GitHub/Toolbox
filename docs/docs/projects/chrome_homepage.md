# Chrome Homepage

This project is a customizable [Chrome Homepage](https://github.com/ALXS-GitHub/Chrome-Homepage) extension that allows users to personalize their new tab page with various favorite links and a background image.

More information on how to install and use the extension can be found in the [GitHub repository](https://github.com/ALXS-GitHub/Chrome-Homepage).
