# Chrome Homepage

[Chrome Homepage](https://github.com/ALXS-GitHub/Chrome-Homepage) is a Chrome extension that replaces the default new tab page with a fully customizable homepage. It lets you organize your favorite links and set a background image, turning the new tab into a personal dashboard rather than Chrome's default shortcuts grid.

The extension is built with HTML, CSS, and JavaScript. It stores your configuration locally in the browser so everything persists across sessions without any external account or server. You can add, reorder, and group links, and pick a background image from a local file or a URL.

To install it, clone the repository, then go to `chrome://extensions`, enable Developer Mode, and click "Load unpacked" pointing to the project folder. From there the new tab page is replaced immediately, and all customization happens directly on the page itself. The full setup instructions and available options are documented in the [GitHub repository](https://github.com/ALXS-GitHub/Chrome-Homepage).
