# RL-Designer

[RL-Designer](https://github.com/ALXS-GitHub/RL-Designer) is a desktop application that allows users to download custom decals for the Alpha Console mod.

## Installation

The latest version of [RL-Designer](https://github.com/ALXS-GitHub/RL-Designer) can be downloaded from the releases page or following this link: [Latest Release](https://github.com/ALXS-GitHub/RL-Designer/releases/latest/).

Just download the `.msi` installer and run it.

Updates will be done automatically through the application.

## Usage

For a complete guide on how to use the application, you can either go in the `HowTo` section of the application, or check the [Documentation on the GitHub repository](https://github.com/ALXS-GitHub/RL-Designer/tree/main/docs).
