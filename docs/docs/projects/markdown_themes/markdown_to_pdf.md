# Markdown to PDF

Conversion md to pdf compatible with [ALXS' Markdown Themes](https://github.com/ALXS-GitHub/Markdown-Themes)

## Usage

The instructions to build the binaries if you want to use the application's binaries are available in the documentation on the [GitHub repository](https://github.com/ALXS-GitHub/Markdown-to-Pdf).

Usable via command line with:
```bash
mdtohtml <filename>
htmltopdf <filename>
mdtopdf <filename>
```

Another way to use the converter is the use directly the `.js` files in the `src` folder as they provide the proper functions to convert markdown to pdf.
