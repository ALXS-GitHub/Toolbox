# Markdown Themes

Custom themes for markdown files made by [ALXS](https://github.com/ALXS-GitHub).

## Getting started

The easy way to use the themes is to use the [VSCode extension](https://github.com/ALXS-GitHub/Markdown-Themes-VSC-Extension). This will allow to have a running server to preview the themes in real-time with the Markdown preview in VSCode. 

You can also inject the theme directly into your markdown files by adding the injection script. See the documentation on the [GitHub repository](https://github.com/ALXS-GitHub/Markdown-Themes) for more details.
