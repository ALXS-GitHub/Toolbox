# Markdown Theme VSC Extension

VSCode Extension linked with the [Markdown Themes](https://github.com/ALXS-GitHub/Markdown-Themes) repository.

## Getting started

To use the themes in the VSCode extension, you need to download the `Markdown Themes` repo, then set up the following environment variable `ALXS_MD_THEME_PATH=<path_to_repo>\Markdown-Themes\themes`, this will allow the extension to find the themes.

This extension also uses the [Markdown-to-PDF repository](https://github.com/ALXS-GitHub/Markdown-to-Pdf) as a submodule. Make sure the submodule is up to date.
