# LazyVim Config

My LazyVim configuration is available on [GitHub](https://github.com/ALXS-GitHub/LazyVim-Config).

In order to use it, just clone the repository contents into your Neovim configuration directory, typically located at `~\AppData\Local\nvim`.

---

Warning: This configuration requires LazyVim `12.44.1` and not any version above `13.x.x` because some new features may break the plugins used. This configuration will probably change in the future to accommodate updates in LazyVim. I do not recommend using this configuration in order to use newer versions of LazyVim.
