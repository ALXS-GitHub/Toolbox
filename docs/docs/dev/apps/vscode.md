# Visual Studio Code

Visual Studio Code is a source-code editor developed by Microsoft for Windows, Linux and macOS. It includes support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

So far this is my favorite code editor, as it is lightweight, fast, highly customizable with a vast library of extensions and it also integrates AI-powered features for enhanced productivity.
