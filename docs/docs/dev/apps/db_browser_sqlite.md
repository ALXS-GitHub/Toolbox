# DB Browser for SQLite

**[DB Browser for SQLite](https://sqlitebrowser.org/)** is a high‑quality, open‑source visual tool for creating, designing, and editing SQLite database files. It provides an intuitive graphical interface to browse tables, run SQL queries, edit records, design schemas, and import/export data (CSV/SQL). It’s useful for developers and analysts who need a lightweight, user-friendly way to inspect and manipulate local SQLite databases on Windows.
