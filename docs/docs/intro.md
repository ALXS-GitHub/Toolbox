# Toolbox

Hey there! Welcome to my personal toolbox documentation. Here, you'll find a curated collection of tools, configurations, and projects that I use regularly. This documentation serves as a reference for myself and anyone interested in exploring the tools I find useful.

All the documentation found here is only part of my experience and own use of the tools, and may not cover every aspect or use case, or may not follow the best way path for using or customizing these tools. This documentation is mainly focused on my personal setup and preferences, and may not be suitable for everyone, but still aims to provide a strong basis for understanding and using these tools effectively.

The toolbox is organized into several sections to make things easier to navigate. The **Apps** section covers desktop applications I rely on daily, from browsers and utilities to entertainment and creative tools. **Dev** contains notes on development tools, editors, languages, and CLI utilities that are part of my workflow. The **AI** section documents AI-related tools and services I experiment with or use regularly. **Projects** gathers documentation around personal projects I have built or maintained. Finally, **Websites** lists web-based tools and services worth bookmarking.

Outside of the main docs, you will also find a dedicated **Games** section with notes on the games I play, including launcher configurations, mod setups, and emulator tweaks, as well as a **Hardware** section covering physical gear and peripherals.
